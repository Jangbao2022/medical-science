物理学术语
感绿胶片（Sense of green film）:
这是一种配合发绿色荧光增感屏使用的胶片，吸收光谱的峰值约为550nm。
自旋(spin ): 
原子核及质子围绕着自身的轴进行旋转。
相位(Phase ):
平面内旋转的矢量与某一参照轴的夹角称为相位。
同相位( in- phase ):
多个矢量在空间的方向一致。
离相位( out of phase ):
多个矢量在空间的方向不-致。
聚相位(re- phase ):
 由不同相位达到同相位的过程。
失相位( de- phase ):
由同相位变成不同相位的过程。
重复时间( repetition time, TR ):
是指从第一个RF激励脉冲出现至下一周期同一脉冲出现时所经历的时间。TR控制着Mz恢复的程度，因而决定着图像的T加权程度。
回波时间(echo time, TE ):
是指第一个RF脉冲到回波信号产生所需要的时间。TE控制着Mxv衰减的程度，因而决定着图像的T2加权程度。
有效回波时间( effective echo time, ETet):
是指在最终图像上反映出来的决定图像对比的回波时间。ETa-般位于回波链的中点，当相位编码梯度的幅度为零或在零附近时，所采集信号的回波时间就是ETeto选用不同的ETet将得出不同的图像对比度。
回波间隔时间( echo train spacing, ETS )
:是指快速自旋回波序列回波链中相邻两个回波之间的时间间隔。ETS决定序列回波时间的长短，关系到图像的对比度。
反转时间( inversion time, TI ):
是指在反转恢复脉冲序列中，180°反转脉冲与90°激励脉冲之间的时间。TI的长短对最终的信号和图像对比度都有很大影响。
翻转角( flipangle )
:是指在RF脉冲的激励下，宏观磁化强度矢量Mo偏离B。方向的角度。
感蓝胶片( Sense of blue film):
是配合发蓝色荧光增感屏使用的胶片，感光乳剂的固有感色是以蓝色为主，不添加色素。其吸收光谱的峰值约为420 nm。
感光中心（sensitivity centre）:
就是在乳剂的制备过程中形成的微量银质点。
感光效应（Photographic effect）
使感光系统(屏-片系统)产生的感光效果称为感光效应(E )。
反差系数( y):
称对比度( contrast)系数。反差系数是指特性曲线直线部分的斜率。
平均斜率(G):
连接特性曲线上指定两点密度( D.. +0.25和Dmia +2.00 )的连线与横坐标夹角的正切值。min
最大密度(Dmax):
 对某种感光材料来说，密度上升到一定程度时，不再因曝光量的增加而上升，此时的密度值称为最大密度( Dmax )。
宽容度(L):
是指特性曲线上直线部分在横坐标上的投影，表示的是正确曝光量的范围。
比特(bit):
是信息量单位。二进制数的一位所包含的信息就是一比特。
量子检出效率 ( detective quantum efficience, DQE )
成像系统的有效量子的利用率。
部分容积效应( partial volume effect ): 
某像素位置上可能有多个不同X线吸收系数的体素存在，该处像素的灰度值往往是多个体素灰度值依其体积所占比例而得的平均灰度值的现象。
移动量( gradation shift, GS )
亦称作灰度曲线平移量，一些CR系统的GS=-1 .44~ 1.44,是利用细微调节以获得最优化密度，改变整幅影像的密度。降低GS值即曲线向右移减小影像密度;反之曲线向左移增加影像密度。
空间频率处理( spatial frequency processing )
指CR系统对空间频率响应的调节，主要用于改变影像的锐利度。
频率等级( frequency rank )
即对空间频率范围的分级。涉及由频率处理所增强的影像频率成份的频带。
频率类型( frequency type ):
用于调整增强系数，控制每一种组织密度的增强程度。
